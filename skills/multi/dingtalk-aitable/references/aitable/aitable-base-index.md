# Base 管理

| 命令 | 用途 | 必读参数 |
|---|---|---|
| `base list` | 列出最近访问的 Base | — |
| `base search` | 按名称搜索 Base | `--query` |
| `base get` | 获取 Base 信息（含 tables 列表） | `--base-id` |
| `base create` | 创建 Base | `--name` |
| `base update` | 更新 Base 名称 | `--base-id` `--name` |
| `base delete` | 删除 Base | `--base-id` |
